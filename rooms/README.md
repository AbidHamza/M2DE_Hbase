# Parcours d'apprentissage - Rooms

Ce dossier contient les 8 rooms qui constituent le parcours d'apprentissage complet sur HBase et Hive.

## Progression recommandée

Suivez les rooms dans l'ordre numérique :

1. **Room 0 : Introduction** - Prise en main de l'environnement
2. **Room 1 : HBase Basics** - Opérations CRUD de base
3. **Room 2 : HBase Advanced** - Versions, filtres, optimisation
4. **Room 3 : Hive Introduction** - Premières requêtes SQL
5. **Room 4 : Hive Advanced** - Jointures, partitions, formats
6. **Room 5 : Intégration HBase-Hive** - Combiner les deux technologies
7. **Room 6 : Cas d'usage réels** - Scénarios pratiques
8. **Room 7 : Projet final** - Projet complet autonome

## Comment travailler dans les rooms

### Structure d'une room

Chaque room contient :
- **README.md** : instructions complètes, rappels théoriques, exercices
- **Vos fichiers de travail** : vous les créez au fur et à mesure

### Fichiers à créer

Dans chaque room, vous devrez créer des fichiers comme :
- `room-X_exercices.md` : documentation de vos exercices
- `room-X_commandes.hbase` ou `.hql` : vos commandes
- `room-X_observations.md` : vos réflexions

Les noms exacts sont indiqués dans le README de chaque room.

### Règles importantes

-  Travaillez uniquement dans les dossiers `/rooms`
-  Créez les fichiers demandés dans chaque room
-  Documentez votre progression
-  Commitez votre travail régulièrement (au moins une fois par room)
-  Ne modifiez pas les fichiers en dehors des rooms

### Validation

Chaque room se termine par une section "Validation" qui liste les critères de réussite. Assurez-vous de tous les remplir avant de passer à la room suivante.

## Aide et support

Si vous êtes bloqué :
1. Relisez attentivement le README de la room
2. Consultez les rooms précédentes pour des exemples
3. Vérifiez la documentation HBase/Hive officielle
4. Testez vos commandes étape par étape
